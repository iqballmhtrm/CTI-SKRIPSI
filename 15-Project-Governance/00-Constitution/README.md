# 00 — Constitution Layer

## Status
Draft (scaffold)

## Purpose
Folder ini memuat **Constitution Layer** repository penelitian: lapisan paling
tinggi dan paling stabil yang menetapkan prinsip, keputusan, dan aturan main.
Lapisan ini berada **di atas** dokumen governance R-00 … R-14 dan menaunginya.

## Pemisahan tiga lapisan
- **Scientific Constitution** (folder ini) — prinsip ilmiah, register keputusan,
  dan aturan governance yang bersifat mengikat dan jarang berubah.
- **Research Governance** (R-00 … R-14 di folder induk) — penurunan metodologis
  dari charter hingga evaluation design.
- **Engineering** (di luar `15-Project-Governance/`) — implementasi teknis;
  diperlakukan sebagai turunan/konteks pendukung, bukan sumber keputusan.

## Isi folder
- `Research_Principles_Document.md` — prinsip ilmiah/desain/evaluasi/governance (template).
- `Research_Decisions_Register.md` — register keputusan penelitian (template tabel).
- `Governance_Rules.md` — aturan repository, LOCK, perubahan, traceability, review (template).

## Aturan
- Seluruh berkas pada tahap ini berupa **template**, belum berisi konten ilmiah.
- Constitution Layer tidak mengubah isi R-00 … R-14; ia menaungi dan merujuknya.
