# Research Governance Repository

## Status
Draft (scaffold)

## Purpose
Folder ini berfungsi sebagai **Single Source of Truth (SSOT)** bagi seluruh
penelitian. Ia memuat dokumen governance yang menata penelitian secara berlapis,
dari Project Charter hingga Evaluation Design, sehingga setiap keputusan dapat
dilacak (traceable) dari fondasi ilmiah hingga rancangan evaluasi.

## Cara membaca folder ini
- Dokumen bernomor `R-00` sampai `R-14` adalah lapisan governance yang berurutan.
- `INDEX.md` memuat daftar seluruh dokumen beserta statusnya.
- `TRACEABILITY.md` memetakan keterhubungan antar dokumen (R-xx → R-yy).
- `PRINCIPLES.md` memuat prinsip kerja dan aturan main penelitian.

## Aturan
- Seluruh berkas pada tahap ini berupa **template (skeleton)**, belum berisi
  konten final.
- Pengisian konten ilmiah dilakukan terpisah, per dokumen, oleh Research Owner.
- Dokumen yang sudah dinyatakan LOCKED tidak diubah tanpa keputusan eksplisit.

## Catatan
Berkas lain yang sudah ada di folder ini (mis. `prompts/`, `roadmaps/`,
`steering/`, dan dokumen checkpoint/eksekusi) tidak termasuk dalam scaffold ini
dan tidak diubah.
