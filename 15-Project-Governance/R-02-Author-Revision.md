# R-02 — Author Revision

## Status
LOCKED (Scientific Governance)

## Purpose
Mencatat keputusan penajaman (revisi) Research Owner atas Problem Domain dan
model konseptualnya, agar penurunan berikutnya konsisten. Dimigrasikan dari
keputusan eksplisit pada sesi R-01.2 (final) dan penegasan karakter penelitian.

## Scope
Revisi pada tataran kerangka konseptual & pendirian metodologis; tidak menambah
konsep ilmiah baru.

## Input
R-01 — Problem Domain.

## Output
Problem Domain & model konseptual ter-revisi (deskriptif-integratif).

## Dependencies
R-01.

## Locked Decisions
1. **Lapisan Problem Domain ditambahkan** sebelum rumusan masalah formal; alur:
   Problem Domain → Research Problem → Objectives → Questions → Theoretical
   Foundation → Research Matrix.
2. **Model konseptual deskriptif-integratif** (selaras karakter *design science*),
   bukan model hubungan kausal.
3. **CTI diposisikan sebagai landasan konseptual** (pemberi konteks pada
   transformasi), **bukan moderator**.
4. **Model linear tanpa feedback loop.**
5. **Decision Support sebagai outcome konseptual tunggal.**
6. **Indikator MTTD/MTTR ditunda** ke tahap spesifikasi & evaluasi (bukan level
   Problem Domain).

## Traceability
- **Predecessor:** R-01. **Successor:** R-03.
- Keputusan 2–5 → diwujudkan pada backbone konsep **R-09**.
- Keputusan 6 → ditindaklanjuti pada **R-14** (Evaluation Design).
